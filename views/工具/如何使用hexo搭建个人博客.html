                    如何使用hexo建立个人博客并部署到Github

                    jyxiaoka 2016-03-07 Hexo

                    hexo是一个快速、简单并且强大的博客框架，它可以不依赖数据库，直接使用markdown语法编写.md格式的文章，然后使用hexo命令迅速生文章的成静态网页(基于Nodejs的强大之处)。几十篇的文章只需几秒即可生成，速度非常快。下面将介绍如何来使用它。

                    # 安装git

                    安装hexo之前需要先安装git和node.js mac下已自带git，使用终端即可。 Windows下需要使用git bash。安装路径：http://git-scm.com/download/。

                    # 安装node.js

                    node.js的安装路径：https://nodejs.org/en/。

                    # 安装hexo

                    安装hexo只需几条命令：

                    $ npm install -g hexo
                    1

                    注意：-g是指全局安装hexo，如果没有-g即安装到某个制定目录 创建Hexo文件夹 使用hexo init Hexo 即会自动在目标文件夹建立网站所需要的所有文件。

                    $ mkdir hexo
$ hexo init
                    1
                    2

                    # 安装依赖包：

                    $ npm install
                    1

                    到此为止已经搭建好本地的hexo博客了。 使用如下命令即可启动博客网站：

                    $ hexo generate #简写hexo g
$ hexo server #简写hexo s
                    1
                    2

                    # 在github上创建Repository

                    创建的时候注意Repository的名字。比如我的Github账号是iosdashen，那么我应该创建的Repository的名字是：iosdashen.github.io。

                    # 设置SSH Keys

                    $ ls -al ~/.ssh #检查是否已经存在了SSH keys
                    1

                    输入指令生成,一路回车即可：

                    $ ssh-keygen -t rsa -C "xxx@163.com"
                    1

                    然后键入以下指令：

                    ssh-agent -s
                    1

                    继续输入指令：

                    ssh-add ~/.ssh/id_rsa
                    1

                    如果你的也是这样子出错了的话，就输入以下指令：

                    eval ssh-agent -s
ssh-add
                    1
                    2

                    输入你的Github密码即可完成SSH Key的添加。嗯，最后还是测试一下吧，键入以下命令：

                    ssh -T git@github.com
                    1

                    # 配置_config.yml

                    在hexo的_config.yml配置文件最后配置github的仓库地址：

                    # Deployment
## Docs: https://hexo.io/docs/deployment.html
deploy:
  type: git
  repository: 你github仓库地址
  message: 'update'
  branch: master
                    1
                    2

                    6
                    7

                    # 生成网站并发布

                    hexo generate //hexo g
hexo deploy   //hexo d
                    1
                    2

                    # 在git使用中遇到的错误

                    1）上传代码时遇到如下错误： 原因是本地中没有README.md文件，所以使用以下命令把README.md文件拉到本地即可。

                    $ git pull --rebase origin master
