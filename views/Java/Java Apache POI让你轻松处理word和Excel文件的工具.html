        Java Apache POI，一个让你轻松处理word和Excel文件的工具

        jyxiaoka 2020-05-09 Java

        “ 一般我们在工作中都无法避免整理Word文件或Excel文件，或程序中有需求需要导出或生成Excel表格、Word文档等，我们可以使用Java POI来帮助我们完成这些繁琐的工作。”

        做过的项目中大多数会选择引入Apache POI jar包，利用它的相关API完成。当然还有JACOB（java -com bridge）这个选择，据说处理表格能力挺强大的，但性能却比POI差一些，所以大部分时候会选择使用POI。

        信心网上有很多网友都写过自己使用POI处理Excel等office文档的程序经验分享出来，也会自己总结一些自己遇到的坑。所以这里也是前人栽树，后人乘凉罢了，自己总结一下，把相关的处理弄成一个工具类，便于后面项目中使用，权当是为了以后出现类似的需求时节省自己的时间吧。

        # 引入POI相关jar包

        这里以使用maven为例，在pom.xm中引入：

          <!-- POI Excel及Word操作API -->

        # 处理Excel

        一般需求是根据数据生成Excel表格：

        /**
   * 通过设置数据和Excel路径生成表格
   * @param header 表头
   * @param cellHeigh 每一列的高度
   * @param data 列表数据
   * @param outPutPath 输出Excel文件路径
   * @throws Exception
   */
  public static void generateExcel(String[] header, int[] cellHeigh, JSONArray data, String outPutPath) throws Exception {
    if (data == null || data.length() == 0) {
      throw new RuntimeException("没有数据，不能执行创建Excel文件任务！");
    }
    HSSFWorkbook workbook = new HSSFWorkbook();
    HSSFSheet sheet = workbook.createSheet("sheet1");
    //是否加密这个sheet
    sheet.protectSheet("pass123456");
    //表格样式、行宽高
    HSSFCellStyle style = workbook.createCellStyle();
    //是否锁住，可以应用在锁定某些单元格
    style.setLocked(true);
    //居中对齐
    style.setAlignment(HorizontalAlignment.CENTER);
    style.setFillBackgroundColor(IndexedColors.DARK_BLUE.getIndex());
    style.setFillForegroundColor(IndexedColors.WHITE.getIndex());
    //style.setFillPattern(FillPatternType.SOLID_FOREGROUND);
    //字体
    HSSFFont font = workbook.createFont();
    font.setBold(true);
    font.setColor(Font.COLOR_RED);
    font.setFontHeightInPoints((short)12);
    style.setFont(font);
    //列宽度自适应
    //sheet.autoSizeColumn(1);
    sheet.autoSizeColumn(1, true);
    
    //设置cell宽度
    for (int i = 0; i < cellHeigh.length; i++) {
      sheet.setColumnWidth(i, cellHeigh[i] * 256);
    }
    
    //合并单元格,此处是合并第0行的0~3列
    //CellRangeAddress rangeAddress = new CellRangeAddress(0, 0, (short)0, (short)2);
    //sheet.addMergedRegion(rangeAddress);
    
    //创建表头行
    if (header != null && header.length > 0) {
      HSSFRow headerRow = sheet.createRow(0);
      headerRow.setHeightInPoints(15);
      for (int i = 0; i < header.length; i++) {
        HSSFCell headerCell = headerRow.createCell(i);
        headerCell.setCellStyle(style);
        headerCell.setCellValue(header[i]);
      }
    }
    //创建数据行
    for (int i = 0; i < data.length(); i++) {
      JSONObject jsonObject = data.getJSONObject(i);
      HSSFRow row = sheet.createRow(i+1);
      Iterator<String> iterator = jsonObject.keys();
      int cellIndex = 0;
      //遍历创建cell
      while (iterator.hasNext()) {
        String key = iterator.next();
        if(jsonObject.has(key) && jsonObject.get(key) != null) {
          HSSFCell cell = row.createCell(cellIndex);
          cell.setCellValue(jsonObject.getString(key));
        }
        cellIndex++;
      }
    }
    //写入文件
    writeToDisk(workbook, outPutPath);
  }
  
  /**
   * 把workbook写到文件中去
   * @param workbook
   * @param diskPath
   * @throws IOException
   */
  private static void writeToDisk(HSSFWorkbook workbook, String diskPath) throws IOException {
    FileOutputStream outStream = new FileOutputStream(new File(diskPath));
    workbook.write(outStream);
    outStream.flush();
    outStream.close();
    workbook.close();
    System.out.println("已成功生成Excel文件！");
  }

        读取表格是这样的：

        /**
   * 从Excel中读取并返回数据
   * @param path
   * @param sheetIndex
   * @return
   * @throws IOException 
   */
  public JSONArray readExcelFromPath(String path, int sheetIndex) throws IOException {
    //读取Excel文件到文件流
    InputStream inputStream = new FileInputStream(new File(path));
    HSSFWorkbook workbook = new HSSFWorkbook(inputStream);
    HSSFSheet sheet = workbook.getSheetAt(sheetIndex);
    JSONArray array = new JSONArray();
    int lastRowNum = sheet.getLastRowNum();
    for (int i = 0; i <= lastRowNum; i++) {
      HSSFRow row = sheet.getRow(i);
      JSONArray rowDataArray = new JSONArray();
      int lastCelNum = row.getLastCellNum();
      for (int j = 0; j <= lastCelNum; j++) {
        HSSFCell cell = row.getCell(j);
        String valueString = cell.getStringCellValue();
        rowDataArray.put(valueString);
      }
      array.put(rowDataArray);
    }
    workbook.close();
    return array;
  }

        可根据自己个性化的需求处理数据。 注意：

        • cell的文本字符容量有限制，保存一些大文本时会报错，此时就需要自己截取一下文本再保存了。
        • HSSFWorkbook可能适合大多Excel，但貌似一些旧版本的officeExcel编写后的Excel读取会出现错误，提示要使用XSSFWorkbook。对应的Row Cell也改成XS前缀的类来完成。

        # 处理Word

        Word当时项目的需求是读取Word文档的所有目录，所以这里只编写了读取目录大纲的代码。其他相关的功能可以在POI API文档中查阅。

        /**
   * 获取Word文档目录层级结构（大纲）
   * @param docPath
   * @throws IOException 
   */
  public static List<String> getDocCategories(String docPath) throws IOException {
    //读入文件流
    InputStream is = new FileInputStream(docPath);
    XWPFDocument document = new XWPFDocument(is);
    List<String> thirdCateList = new ArrayList<String>();
    //获取段落
    List<XWPFParagraph> paragraphs = document.getParagraphs();
    
    for (XWPFParagraph paragraph : paragraphs) {
      // 读取段落标题
      String title = paragraph.getParagraphText();
      // 读取目录层级，从1开始
      String style = paragraph.getStyle();
      if (title != null && style != null) {
        title = unescapeJava(title);
        if ("1".equalsIgnoreCase(style)) {
          thirdCateList.add("一级标题：" + title);
        }
        if ("2".equalsIgnoreCase(style)) {
          thirdCateList.add("二级标题：" + title);
        }
        if ("3".equalsIgnoreCase(style)) {
          thirdCateList.add("三级标题：" +title);
        }
      }
    }
    document.close();
    return thirdCateList;
  }
